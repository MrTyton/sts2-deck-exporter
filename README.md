# Slay the Spire 2 Deck Exporter

A web application to visualize and export your decks from Slay the Spire 2. 

## Features
- Upload your Slay the Spire 2 `.run` save files
- Visualize your current deck, including all cards and upgrades
- Filter decks by card type, rarity, and energy cost
- Export your deck visualization to an image for easy sharing with friends or the community

## Installation & Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/MrTyton/STS2-Deck-Exporter.git
   cd STS2-Deck-Exporter
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
